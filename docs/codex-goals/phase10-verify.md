# Phase 10 Verify

1. `cargo run --release` で gototerm を起動する。
2. `Ctrl+Shift+F` を押し、左が files/changes 一覧、右上がプレビュー、右下が従来のターミナル領域になることを確認する。
3. 左の files 一覧でファイルを選び `Enter` を押す。右上プレビューに内容が表示され、右下ターミナルの幅が変わらないことを確認する。
4. 右下ターミナルで `for i in (seq 20); echo "line $i" >> phase10-demo.txt; sleep 0.2; end` を実行し、右上プレビューが追従表示されることを確認する。
5. 左の files または changes 一覧で別ファイルを選び、右上プレビューがそのファイルにピン留めされることを確認する。
6. 左サイドバーにフォーカスした状態で `e` を押す、または右上プレビューの `[編集: nvim]` をクリックし、右上プレビュー枠がエディタに切り替わることを確認する。
7. エディタで内容を編集して `:wq` で終了し、右上が Reader に戻って編集後の内容が表示されることを確認する。
8. もう一度 `Ctrl+Shift+F` を押し、全面ターミナルに戻ることを確認する。
9. ワークベンチ OFF の状態で、タブ、分割、yazi、IME が従来どおり動くことを確認する。
