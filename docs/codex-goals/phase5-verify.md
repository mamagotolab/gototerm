# Phase 5 手動確認手順

1. `gototerm` を起動し、`Ctrl+Shift+F` でサイドバーを表示する。
2. 端末内の出力に表示されたファイルパスをクリックし、そのファイルがサイドバーでプレビューされることを確認する。
3. プレビュー見出し直下の `[クリックで編集: nvim]` をクリックする。
4. 画面が上下分割され、下ペインで `nvim` が該当ファイルを開いた状態になることを確認する。
5. `:wq` で `nvim` を終了し、エディタペインが閉じて元のレイアウトに戻ることを確認する。
6. `Ctrl+Shift+E` と `Ctrl+Shift+O` の既存分割を実行し、従来どおり新しいシェルが開くことを確認する。
