# Phase 1 手動確認手順

1. `cargo run` で gototerm を起動する。
2. `Ctrl+Shift+F` を押し、右サイドバーが表示されることを確認する。
3. サイドバーに現在の作業フォルダ、Git ブランチと staged / modified / untracked / deleted 件数、`claude` / `codex` / `gemini` の有無が表示されることを確認する。
4. もう一度 `Ctrl+Shift+F` を押し、サイドバーが消えて端末領域が全幅に戻ることを確認する。
5. サイドバー表示中と非表示中の両方で、タブ作成・タブ切替・縦横分割・フォーカス移動・ウィンドウリサイズが従来どおり動くことを確認する。
