# Phase 6 手動確認手順

1. gototerm を起動し、サイドバーを表示する。
2. workspace/git 行の下にある切替行をクリックし、`files` モードに切り替わることを確認する。
3. `docs/` や `src/` などのフォルダ行をクリックし、そのフォルダへ潜れることを確認する。
4. `../` をクリックし、親フォルダへ戻れることを確認する。
5. `Cargo.toml` や `README.md` などのファイル行をクリックし、既存のピン留めプレビューに内容が表示されることを確認する。
6. プレビュー上部の `[クリックで編集: nvim]` 行をクリックし、nvim が別ペインで開くことを確認する。
7. エントリ数が多いフォルダで、サイドバー内にカーソルを置いてマウスホイールを回し、長い一覧をスクロールできることを確認する。
8. 切替行をクリックして `changes` モードへ戻し、従来の changed files 表示とプレビューが表示されることを確認する。
9. ネットワーク上の巨大フォルダを `files` モードで開き、一覧読込中も画面操作や描画が固まらないことを確認する。
