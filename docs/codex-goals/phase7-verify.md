# Phase 7 手動確認手順

1. `cargo run` で gototerm を起動する。
2. `Ctrl+Shift+F` でサイドバーを開き、最初から `files` 一覧が表示されることを確認する。
3. `README.md` をクリックする。
4. サイドバーが Reader 幅（既定 `preview_ratio = 0.50`）に広がり、全面プレビューになることを確認する。
5. Markdown の見出しが太字色付き、本文が折り返し表示、ホイールで上下スクロールできることを確認する。
6. `← 戻る` をクリックし、files 一覧と元の幅（`sidebar_ratio`）に戻ることを確認する。
7. もう一度 `README.md` を開き、`[編集: nvim]` をクリックしてエディタペインが開くことを確認する。
8. `[OSの既定アプリで開く]` をクリックし、OS の既定アプリでファイルが開くことを確認する。
9. `changes` モードへ切り替え、変更ファイルの下部ライブ追従プレビューが従来どおり残っていることを確認する。
10. `config.toml` の `editor` に存在しないコマンドを設定して起動し、Reader の `[編集]` クリックで「編集コマンドが見つかりません」という案内が表示されることを確認する。
