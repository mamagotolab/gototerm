# Phase 9 手動確認手順

1. `gototerm` を起動する。
2. `Ctrl+Shift+B` を押し、右サイドバーが表示され、選択行が反転表示されることを確認する。
3. files 一覧で `↓` を連打し、`… ほか N 件` の先にあるフォルダやファイルまで選択が進み、自動スクロールすることを確認する。
4. フォルダを選択して `Enter` を押し、そのフォルダへ潜ることを確認する。
5. `Backspace` を押し、親ディレクトリへ戻ることを確認する。
6. ファイルを選択して `Enter` を押し、Reader が開くことを確認する。
7. Reader で `↑` / `↓` / `PageUp` / `PageDown` / `Home` / `End` を押し、本文がスクロールすることを確認する。
8. Reader で `e` を押し、設定されたエディタ（例: `nvim`）が新しいペインで開くことを確認する。
9. `Esc` を押し、フォーカスが端末へ戻ることを確認する。
10. 端末で通常の文字入力がシェルへ届くことを確認する。
11. もう一度 `Ctrl+Shift+B` でサイドバーへフォーカスし、文字キーを押してもシェルへ入力されないことを確認する。
12. サイドバー内をクリックするとサイドバーへフォーカスし、端末ペインをクリックすると端末へフォーカスが戻ることを確認する。
13. `Ctrl+Shift+F` でサイドバーを非表示にしたとき、端末へフォーカスが戻ることを確認する。
14. サイドバーフォーカス中でも既存の `Ctrl+Shift+T/W/Q/E/O/Tab/矢印/F` が従来どおり動くことを確認する。
